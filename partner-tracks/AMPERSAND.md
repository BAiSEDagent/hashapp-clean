# Ampersand

## Why it fits Hashapp
Ampersand is useful if it becomes the communication backbone between a linked agent, the app, and vendor/service actions.

## Product role
Potential uses:
- purchase request transport
- approval / response loop
- agent-to-service coordination

## Product value
Makes Hashapp feel more like a living agent system instead of a static wallet UI.

## Track alignment
- Agents that cooperate
- Agents that trust

## Rule
Only include Ampersand if it is functionally central to the request / approval loop.
