# PRODUCT

## One-line pitch
The safest way to let agents buy things for you.

## Short pitch
Hashapp is a consumer-grade control center for agent spending on Base. Humans can fund agents, set policy-bound budgets, approve or auto-approve purchases, and verify every action through clean receipts and onchain proof.

## Big idea
A human control plane for autonomous commerce.

## Who it is for
- people using AI agents to buy tools or services
- builders coordinating multiple agents
- users who want automation without surrendering wallet control

## Core promise
Let agents buy what they need without losing control.

## MVP loop
1. Connect wallet
2. Fund agent budget
3. Create spending rule
4. Agent requests purchase
5. Human approves or rule auto-approves
6. Payment executes
7. Receipt appears in activity feed
8. Out-of-policy request is blocked

## Design principles
- build for the human, not the agent
- make complex systems legible
- trust through clarity
- consumer polish over crypto dashboard aesthetics
- one real loop beats broad surface area
